📌 Project Overview
The Sales Performance Dashboard is an interactive Excel-based data analytics project designed to monitor and analyze sales performance across different regions and sales executives. It provides valuable insights through dynamic charts, KPIs, and visual reports, enabling data-driven decision-making.
🚀 Features
Interactive dashboard with filters and slicers
Region-wise sales performance analysis
Sales Executive performance tracking
Total Sales calculation and monitoring
Target Achievement (%) analysis
Away From Target (%) measurement
Dynamic charts and visualizations
Easy-to-understand KPI reporting
🛠️ Tools & Technologies
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers
Conditional Formatting
Excel Functions & Formulas
📈 Key Metrics
Total Sales
Target Sales
Target Achievement (%)
Away From Target (%)
Regional Sales Comparison
Top Performing Sales Executives

The dataset includes:
Employee Code

Away From Target Percentage
🎯 Objective
The objective of this project is to transform raw sales data into meaningful insights, helping organizations track performance, identify trends, evaluate targets, and improve business decisions through data visualization.
📊 Dashboard Insights
Compare sales performance across regions.
Identify top and low-performing sales executives.
Monitor target achievement rates.
Analyze sales trends using interactive charts and filters.
